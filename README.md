# 自考必过舱

上海自考 · 上海电机学院 · 数字媒体艺术（专升本）个人备考 PWA。

## 功能

- 科目：`15040` 习近平新时代中国特色社会主义思想概论、`15043/03708` 中国近现代史纲要、`15044/03709` 马克思主义基本原理、`14265` 数字媒体艺术概论、`13511` 多媒体技术与应用。
- 刷题：单选、多选、简答、论述，支持即时判题、参考要点、解析和错题收集。
- 交卷报告：整卷折算分、是否过线、题型得分、薄弱章节和错题回跳。
- 错题复盘：按科目、题型、章节筛选错题，显示累计失分和优先复盘章节，可按当前筛选生成错题重刷卷。
- 学习进度：每门课显示判题进度、已交卷数、待复盘题数、待导入真题和最近一次分数。
- 本课下一步：自动排出继续未完成卷、复盘错题、开刷新卷、补齐最新真题的学习队列。
- 真题记录：历年真题时间轴显示进行中、上次分数和待复盘题数。
- 题库：内置公共课、数媒概论与计算机重点训练卷，公共课、数媒概论和计算机训练卷各 10 题并覆盖客观题、简答、论述；历年真题先放索引，支持通过 JSON 导入本地整理后的真题卷。
- 资源矩阵：按科目列出可刷卷、待补真题、资料入口和检索词，可一键预填粘贴制卷。
- 粘贴制卷：从网页、PDF OCR 或 Word 复制题目文本，先预览题型、答案、解析缺口，可手工校正答案和解析后再确认导入。
- 真题入口：在“历年真题”点待导入卷，会自动切到资源页并预填制卷信息。
- 导入卷管理：资源页可查看补齐进度、开刷本地导入卷、校正已导入答案/解析、导出备份、删除单套导入卷。
- 补齐状态：本地导入卷会自动覆盖同科目、同年份、同考期的待补索引，真题时间轴不会重复显示同一套卷。
- AI 评价：设置页填写 DeepSeek API Key 后，简答和论述题可请求 AI 评价；未配置时使用本地关键词估分。
- PWA：可安装到桌面或手机主屏，基础资源支持离线打开。

## 开发

```bash
pnpm install
pnpm dev
```

常用检查：

```bash
pnpm lint
pnpm build
```

## 真题导入

推荐先用应用“资源”页的“粘贴文本制卷”：

```text
一、单项选择题
1. 题干（ ）
A. 选项
B. 选项
C. 选项
D. 选项

2. 题干（ ） A. 选项 A B. 选项 B C. 选项 C D. 选项 D

参考答案：
1.B 2.C

答案及解析：
1.B 解析：这里写解析
2.C 解析：这里写解析
```

也支持每题后面直接写 `答案：B`、`解析：...` 的格式。简答题和论述题如果答案集中写在卷尾，也可以按题号写：

```text
5. 答案：第一要点；第二要点；第三要点。解析：这里写易错点。
第6题：先写总观点；再分层展开；最后结合材料。解析：论述题要有结构。
```

如果已经整理成结构化数据，也可以在“资源”页下载 `zikao-import-template.json`，按模板填写：

- `papers`: 试卷元信息，`questionIds` 对应本卷题目 ID。
- `questions`: 题目数据，支持 `single`、`multiple`、`short`、`essay`。

导入后的真题只保存在浏览器本地，不会提交到仓库。

资源页的“备份全部导入卷”会下载当前浏览器里的本地题库 JSON。换电脑或清浏览器数据前，先导出备份。

## 资料口径

第一版内置题目主要依据官方考试计划和大纲进行训练化整理，高频强化卷属于模拟训练题，不冒充历年真题。完整历年真题多来自第三方资料站或个人资料包，版权状态不统一，因此仓库只放索引和导入机制；下载到本地的资料可以继续整理成 JSON 后导入。
